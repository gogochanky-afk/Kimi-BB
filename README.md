# Kimi-BB - Hugo Sammie Family Office Automation

## Setup Instructions

1. Go to Settings > Secrets and variables > Actions
2. Add these 3 secrets:
   - TELEGRAM_BOT_TOKEN
   - TELEGRAM_CHAT_ID (7042296651)
   - TELEGRAM_GROUP_ID (-5153249366)

3. Daily reports will run at:
   - 8:30 AM (premarket)
   - 10:00 AM (postmarket)

4. Manual run: Actions > Daily Market Report > Run workflow
